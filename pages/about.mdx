Gun violence is a growing epidemic across the U.S., and Americans by-and-large [support gun control](https://www.vox.com/2015/10/3/9444417/gun-violence-mass-shootings-us-america), yet Congress refuses to pass it as legislation. Meanwhile, the gun lobby is directly funding our representatives: the current members of Congress have accepted [$40M from gun rights](https://www.opensecrets.org/industries/summary.php?ind=Q13&recipdetail=A&sortorder=U&cycle=All) groups and [about $6M from gun control](https://www.opensecrets.org/industries/summary.php?ind=Q12&recipdetail=A&sortorder=U&cycle=All) groups (about 2/3 of which since 2016).

## The data

The data all comes from the [Center for Responsive Politics](https://www.opensecrets.org), which gets it directly from the filings every Congressperson is required to file with the federal government listing their funders.

## The site

This project debuted at the [NYU ITP/IMA Winter Show 2019](https://itp.nyu.edu/shows/winter2019/). It runs on [React](https://reactjs.org), [Next.js](https://nextjs.org), and [Theme UI](http://theme-ui.com), hosted on [Now](https://zeit.co). The code is entirely [open source](https://github.com/lachlanjc/gunfunded).
